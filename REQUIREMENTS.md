# RalphGrip - AI 기반 프로젝트 관리 도구

## 프로젝트 개요

Jira의 기능에 AI를 결합한 프로젝트 관리 도구. Notion의 과도한 자유도와 Jira의 AI 부재 문제를 해결하고, Slack/Notion 연동을 통해 효율적인 프로젝트 관리를 지원한다.

---

## 핵심 요구사항

### 1. 인증

- 구글 로그인 지원

### 2. 작업 단위 구조

- **Epic**: 대규모 기능 단위
- **Story**: Epic 하위의 사용자 스토리
- **Ticket**: 개별 작업 단위

각 작업 단위에 대해:
- 속성 지정 가능
- 기본 속성 제공 (상태, 우선순위, 담당자, 기한 등)
- 담당자 할당 기능

### 3. 프로젝트 관리

- 프로젝트 단위로 작업 관리
- 메인페이지에서 프로젝트 선택/전환
- 프로젝트 페이지 템플릿 제공
  - 필수 정보 입력을 강제하여 습관화
  - 새로운 사람도 프로젝트 현황을 한눈에 파악 가능
  - 표준화된 구조로 정보 누락 방지

### 4. 뷰 타입

- **칸반보드**: 상태별 카드 이동
- **리스트뷰**: 테이블 형식 조회
- **타임라인뷰**: WBS/마일스톤 시각화
- **Mine 뷰**: 개인별 할당된 태스크 모아보기
  - 폴더 구조로 관리되는 프로젝트에서 태스크 생성 시 자동 폴더 귀속
  - 고아 Task 방지 (어느 폴더에도 속하지 않는 태스크 생성 방지)

모든 뷰에서 상태 변경 가능

### 5. 관리자 뷰

- 팀원별 업무 진행현황 모니터링
- 태스크 할당자 관점의 트래킹
  - 내가 준 태스크들의 진행 상황 추적
- 개인별 할일(Todo) 리스트 조회
  - Jira처럼 한눈에 보이는 형태
- 개인별 기간별 프로젝트 집중도 표시
  - 어느 기간에 어떤 프로젝트에 집중하는지 시각화

### 6. 이슈 관리

- 프로젝트 일정에 영향을 주는 주요 이슈 한눈에 보기
- 각 이슈별 대처 결정사항 기록
- 우선순위 및 긴급도 태그
- 시각적으로 인지하기 쉬운 UI (색상, 아이콘 등)

### 7. 공수 관리

- Man hour 기입 기능
- 공수 통계 조회
  - 프로젝트별
  - 담당자별
  - 기간별

### 8. AI 기능

- Knowledge 기반 AI 어시스턴트
- 프로젝트 컨텍스트 이해
- 작업 추천, 요약, 분석 기능

### 9. 외부 연동

- Slack API 연결
  - 알림
  - 상태 업데이트
- Notion 연동 (선택)
- Google Calendar 연동
  - 개인별 캘린더와 태스크 일정 통합 조회
  - 프로젝트 일정과 개인 일정 함께 보기
